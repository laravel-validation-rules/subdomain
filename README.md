Supports Laravel: 5.5, 5.6, 5.7, 5.8, & 6

## Installation

```bash
composer require laravel-validation-rules/subdomain
```

## Usage

```php
use LVR\Subdomain\Subdomain;

$request->validate([
    'domain' => ['required', new Subdomain],
]);
```
